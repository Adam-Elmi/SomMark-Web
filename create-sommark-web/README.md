# create-sommark-web

Set up a new SomMark Web project instantly.

## Usage

```bash
npx create-sommark-web@latest
```

Follow the interactive prompts to set up your project.

## What's included?

- Full Vite configuration for SomMark.
- Ready-to-use `index.smark` shell template.
- File-based routing setup in `src/pages`.
- High-performance SSG defaults.
